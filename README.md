## üöÄ Key Features

* **Home Page:** A visually striking landing page with a neon audio visualizer, 3D tilt-effect cards for team members, and a dynamic event grid.
* **Booking System:** An interactive form that captures user details and ticket preferences, using LocalStorage to pass data between pages.
* **Digital Ticket:** A dynamically generated ticket that displays the user's name, selected artist, and randomly assigns a unique **Seat Number** and **Venue** in Islamabad.
